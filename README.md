# WeatherApp

It's a sample app code of Weather app.

## How to use

1. To run the project
  
  ```
  npm install
  ```
  
  ```
  npx pod-install
  ```

2. For android 
```
npm run android 
```
or
```
npx react-native run-android
```

3. For ios
```
npm run ios
```
or
```
npx react-native run-ios
```
